# 🩸 抖音直播血条插件（开箱即用版）

一个用于抖音直播的实时血条插件。根据观众送出的礼物实时增减主播的血条，适合游戏直播互动场景。
**已集成 douyinLive 服务，程序启动自动运行，无需任何额外环境。**

---

## 🚀 开箱即用（推荐）

1. 到 [GitHub Release](https://github.com/MIAO-robot/blood-bar/releases) 下载 `抖音血条插件-win32-x64.zip`
2. 解压后双击 `抖音血条插件.exe`
3. **右键血条 → 配置面板**，填入 Cookie 和直播间 ID
4. 保存，开始直播！

> 下载的是一个**完整独立程序**，自带 Electron 运行环境和 douyinLive 服务，
> **不需要安装 Node.js、不需要 npm、不需要手动启动任何服务**。

---

## ✨ 功能特性

- 🔗 **实时礼物监听** — 内置 douyinLive 服务，自动连接直播间获取礼物数据
- ❤️ **血量增减** — 4 个加血礼物 + 4 个减血礼物，共 8 个槽位
- ⚙️ **灵活配置** — 每个礼物的血量变化数值可自由设置
- 🎨 **自定义外观** — 支持自定义血条内框/外框图片
- 🖼️ **默认百搭样式** — 自带金属质感边框 + 深色网格背景
- 🖱️ **右键配置** — 右键血条即可打开配置面板
- 📌 **置顶无边框** — 透明无边框窗口，始终置顶，适合 OBS 叠加

## 📋 使用步骤

### 第一步：获取抖音 Cookie

1. 浏览器打开 https://live.douyin.com 并登录
2. 按 `F12` 打开开发者工具 → `Network` 标签
3. 点击任意请求，复制请求头中的完整 `Cookie` 值

### 第二步：配置插件

1. 右键血条 → 选择「配置面板」
2. 填入 **Cookie** 和 **直播间 ID**（抖音直播 URL 中的数字）
3. 配置礼物和血量（详见下方）
4. 保存配置，血条立刻开始工作

### 第三步：配置礼物

- **加血槽位 (1-4)**：收到这些礼物时增加血量
- **减血槽位 (5-8)**：收到这些礼物时减少血量
- **礼物ID**：从 douyinLive 控制台输出中获取（送一个礼物后查看 `gift_id`）
- **血量值**：每次收到该礼物变化的血量（加血为正，减血为负，连击会叠加）

## 🎮 使用技巧

- **OBS 叠加**：OBS 中添加「窗口捕获」→ 选择血条窗口 → 勾选「允许透明」
- **调整大小**：直接拖拽血条窗口边缘
- **快速重置**：右键血条 → 「重置血条」
- **获取礼物ID**：送一个礼物后，查看 douyinLive 控制台输出的 `gift_id` 字段
