# 🩸 抖音直播血条插件

一个用于抖音直播的实时血条插件。根据观众送出的礼物实时增减主播的血条，适合游戏直播互动场景。

## ✨ 功能特性

- 🔗 **实时礼物监听** - 通过 douyinLive 服务连接抖音直播间，实时获取礼物数据
- ❤️ **血量增减** - 支持 4 个加血礼物 + 4 个减血礼物，共 8 个槽位
- ⚙️ **灵活配置** - 每个礼物的血量变化数值可自由设置
- 🎨 **自定义外观** - 支持自定义血条内框/外框图片
- 🖼️ **默认百搭样式** - 自带金属质感边框 + 深色网格背景，适配各种直播风格
- 🖱️ **右键配置** - 右键血条即可打开配置面板
- 📌 **置顶无边框** - 透明无边框窗口，始终置顶，适合 OBS 叠加

## 📋 前置要求

1. **Node.js** (v18+) - [下载地址](https://nodejs.org/)
2. **douyinLive 服务** - 用于连接抖音直播间获取礼物数据

## 🚀 快速开始

### 第一步：安装 douyinLive 服务

douyinLive 是一个开源的抖音直播数据抓取工具，负责连接抖音并转发礼物消息。

**方式一：下载预编译版本（推荐）**
1. 访问 [douyinLive Releases](https://github.com/jwwsjlm/douyinLive/releases)
2. 下载对应系统的版本（Windows 选 `windows-amd64.zip`）
3. 解压后运行 `douyinLive.exe`，默认监听 `1088` 端口

**方式二：Docker 运行**
```bash
docker run -d --name douyinlive --restart unless-stopped -p 1088:1088 ghcr.io/jwwsjlm/douyinlive:latest
```

### 第二步：获取抖音 Cookie

1. 浏览器打开 https://live.douyin.com 并登录
2. 按 `F12` 打开开发者工具
3. 切换到 `Network` 标签
4. 点击任意请求，复制请求头中的完整 `Cookie` 值

### 第三步：启动血条插件

**Windows 用户：** 双击 `启动血条插件.bat`

**Mac/Linux 用户：**
```bash
npm install
npm start
```

### 第四步：配置插件

1. 右键点击血条 → 选择「配置面板」
2. 填入 Cookie 和直播间 ID（抖音直播URL中的数字）
3. 配置礼物和血量
4. 保存配置，开始直播！

## ⚙️ 配置说明

### 连接设置
| 参数 | 说明 |
|------|------|
| Cookie | 抖音直播的 Cookie，用于鉴权 |
| 房间号 | 直播间 URL 中的数字 ID |
| WS端口 | douyinLive 服务的端口，默认 1088 |

### 礼物配置
- **加血槽位 (1-4)**：收到这些礼物时增加血量
- **减血槽位 (5-8)**：收到这些礼物时减少血量
- **礼物ID**：从 douyinLive 控制台输出中获取
- **血量值**：每次收到该礼物变化的血量（加血为正数，减血为负数，连击会叠加）

### 外观设置
- **内框图片**：血条背景图，会显示在血量填充层下方
- **外框图片**：血条边框图，覆盖在最上层

> 💡 程序自带默认的金属质感外框和深色网格内框，无需设置即可使用。

## 🎮 使用技巧

1. **OBS 叠加**：在 OBS 中添加「窗口捕获」，选择血条窗口，勾选「允许透明」
2. **调整大小**：直接拖拽血条窗口边缘调整大小
3. **快速重置**：右键血条 → 选择「重置血条」
4. **获取礼物ID**：保持 douyinLive 控制台可见，送一个礼物后从输出中找到 `gift_id`
